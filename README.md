# ⚜️ Vinclub - Hệ Sinh Thái Đầu Tư Thượng Lưu

Hệ thống quản lý và đồng bộ thời gian thực (Realtime) giữa **Admin Panel** và **User App**, tối ưu hóa trải nghiệm đầu tư tài sản cao cấp và ký kết hợp đồng ủy thác.

## 🚀 Tính Năng Cốt Lõi
* **User App:** Giao diện tối sang trọng (Charcoal/Gold), hiệu ứng xếp chồng xòe bài mượt mà khi duyệt dự án, luồng cấu hình hạn mức và ký hợp đồng trực quan.
* **Admin Panel:** Theo dõi trạng thái người dùng realtime, phê duyệt lệnh nạp/rút an toàn tài chính, cấu hình động thông số dự án và phát tin tức khẩn cấp.

## 🛠️ Công Nghệ Sử Dụng
* **Frontend:** React, TypeScript, Tailwind CSS, Vite
* **Realtime Database:** Firebase (Firestore / Realtime Database) / Supabase
